== Description ==


Post Timeline is the WordPress Plugin that creates Beautiful Timeline Page of your WordPress Posts, using this plugin users can create unlimited beautiful Vertical timelines, user can set their own dates for each post of the timeline, that appears in ASC order by user selected Dates.

<strong><a href="https://posttimeline.com/">DEMO OF TIMELINE</a></strong>

https://www.youtube.com/watch?v=Mk3414Zsl7w

Shortcode [post-timeline id="PARENT_ID_OF_TIMELINE"] to create Post Timeline on Page or Post.


= Post Timeline Features =

* Unlimited Free timelines, with Header Post
* Responsive UI that looks cool on every single device
* Beautiful UI of the vertical timeline
* Add dates for each Post of the timeline
* Multi Language supported


HOW TO USE IT
[post-timeline id="123"]

Where id = 123 is the timeline ID OF THE parent post, which can have unlimited child posts which will appear in timeline.

== Installation ==

Installation of this plugin is pretty easy.

1. Upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation, or upload it directly from your plugins section in the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!

== Frequently Asked Questions ==

= Getting Page Not Found on Read More button =

Goto Settings -> Permalinks -> Select "Post name" Name -> Click Save Changes

If Post name is already selected just click "Save Changes"

= How can i use this plugin? =

1. To show timeline on a Page or Post, Click on Post Timeline and Add New.
2. Add a Post this Post will be your top Post and you have to select this post as parent when creating all timeline Posts.
3. Add a New Post and select previous Post as Parent.
4. Make sure to fill Dates, Position and Sub-Heading when creating posts.

Create a ticket in support for further queries.


= Why my Post Timeline is showing on Right Side of page with scroll bar? =

Timeline works with 100% space on page, when you will make your container 100% with css, the timeline will appear properly.
